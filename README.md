
# 📱 QR Scanner Web App

A modern, lightweight QR Code Scanner built using JavaScript and the browser camera API.  
It allows users to scan QR codes in real-time directly from their device — no backend required.

---

## 🚀 Features

- 📷 Real-time QR code scanning using camera  
- ⚡ Fast and lightweight  
- 🌐 Works directly in the browser  
- 🎯 Clean and modern UI  
- 🔗 Supports URL detection  
- 📱 Mobile-friendly  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6 Modules)  
- qr-scanner library

## ⚙️ Setup Instructions

1. Clone the repository:
git clone https://github.com/anmol29012023-sudo/qr-scanner-app.git

    cd qr-scanner-app


2. Run a local server:

### Using Node.js
    npx live-server


3. Open in browser
 
4. Allow camera access when prompted

---

## 📸 How It Works

- The app accesses your device camera  
- Scans frames in real-time  
- Detects QR codes  
- Displays decoded result instantly  

---

## ⚠️ Important Notes

- Camera access requires **HTTPS or localhost**  
- Make sure both files are present:
  - `qr-scanner.min.js`
  - `qr-scanner-worker.min.js`
